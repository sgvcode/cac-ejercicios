18. Realizá un programa que permita al usuario ingresar 12 valores, de a uno por vez, que representan los sueldos mensuales que percibió un empleado durante un año. Si durante la carga de los sueldos mensuales se detecta un valor negativo, esto indica que aún no se ha cobrado el mes en curso, por lo tanto, deben dejar de ingresarse datos. La computadora debe mostrar el sueldo anual percibido (total o parcial).

        
        
function calcularSueldoAnual() {
  let sueldosMensuales = [];
  let sueldoTotal = 0;

  for (let i = 1; i <= 12; i++) {
    let sueldo = parseFloat(prompt(`Ingrese el sueldo del mes ${i}: `));
    if (sueldo < 0) {
      break;
    } else {
      sueldosMensuales.push(sueldo);
      sueldoTotal += sueldo;
    }
  }

  alert(`El sueldo anual percibido es de $${sueldoTotal.toFixed(2)}`);
}

calcularSueldoAnual();

19. Realizá un programa que permita la validación del ingreso a un sitio web. Teniendo ya precargados un nombre de usuario ("admin") y una contraseña ("123456"), el programa debe permitir al usuario ingresar sus credenciales. Si las mismas son erróneas, se volverán a pedir hasta un máximo de 3 intentos. Finalmente, la computadora debe mostrar alguno de los siguientes mensajes según sea el caso: "Acceso concedido" o "Se ha bloqueado la cuenta".

        
        
function capturarLogin() {
  let username = "";
  let password = "";

  for (let i = 3; i >= 1; i--) {
      username = prompt("Nombre de Usuario: ");
      password = prompt("Password: ");
      if (username !== "admin" || password !== "123456") {
          if (i === 1) {
              alert("Se ha bloqueado la cuenta");
          } else {
              alert(`Credenciales incorrectas. Intentos restantes: ${i - 1}`);
          }
      } else {
          alert("Acceso concedido");
          break;
      }
  }
}
capturarLogin();

21. Realizá un programa que permita al usuario ingresar un número de minutos. La computadora debe mostrar un cronómetro con el formato "mm:ss"desde "00:00"hasta un segundo antes del minuto ingresado.

00:00

        
        
// Pedimos al usuario que ingrese los minutos
const minutos = parseInt(prompt("Ingresa la cantidad de minutos:"));

// Convertimos los minutos a segundos
const segundosTotales = minutos * 60;

// Función que actualiza el cronómetro
function actualizarCronometro(segundosRestantes) {
  // Calculamos los minutos y segundos actuales
  const minutosActuales = Math.floor(segundosRestantes / 60);
  const segundosActuales = segundosRestantes % 60;

  // Mostramos el tiempo actual en formato mm:ss
  console.log(`${minutosActuales.toString().padStart(2, "0")}:${segundosActuales.toString().padStart(2, "0")}`);

  // Si quedan segundos, programamos la siguiente actualización
  if (segundosRestantes > 0) {
    setTimeout(() => {
      actualizarCronometro(segundosRestantes - 1);
    }, 1000);
  }
}

// Iniciamos el cronómetro
actualizarCronometro(segundosTotales);
